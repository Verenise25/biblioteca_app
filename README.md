# Biblioteca App

## Descripción
Aplicación sencilla desarrollada en Python que permite gestionar libros y préstamos.
El proyecto utiliza una estructura modular separando modelos, servicios y el archivo principal.

## Tecnologías utilizadas
- Python 3
- Visual Studio Code
- GitHub

## Conceptos aplicados
- Tipos de datos: string, integer, float, boolean
- Identificadores en snake_case
- Programación orientada a objetos
- Organización del código en módulos

## Ejecución del programa
Desde la carpeta del proyecto ejecutar:

python main.py
